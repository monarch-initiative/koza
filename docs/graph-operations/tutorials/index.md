# Tutorials

Learn graph operations through hands-on, step-by-step lessons. These tutorials are designed for newcomers and guide you through complete workflows from start to finish.

## Available Tutorials

### [Build Your First Graph](first-graph.md)

**Level**: Beginner | **Time**: 15 minutes

Learn the basics of graph operations by creating a simple knowledge graph from KGX files. You'll learn to:

- Join node and edge files into a database
- Explore your graph with SQL queries
- Generate statistics and reports
- Export to different formats

### [Complete Merge Workflow](merge-pipeline.md)

**Level**: Intermediate | **Time**: 30 minutes

Master the full graph merge pipeline by combining data from multiple sources. You'll learn to:

- Prepare input files and SSSOM mappings
- Run the merge pipeline with customization
- Understand pipeline output and diagnostics
- Handle common issues and edge cases

## Prerequisites

Before starting these tutorials, ensure you have:

- Koza installed (`pip install koza`)
- Basic familiarity with the command line
- Sample KGX files (provided in tutorials or from your own data)
